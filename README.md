 Alina Martin - Personal Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, education, and contact information. Built with HTML, CSS, and JavaScript featuring a beautiful sliding page navigation.

 Features
- Sliding Page Navigation- Smooth transitions between sections with Next/Back buttons
- Responsive Design- Works perfectly on desktop, tablet, and mobile devices
- Modern UI- Clean design with Lavender (#C8B6FF) and Deep Purple (#6A4C93) color scheme
- CV Download- Easy access to download my CV from multiple locations
- Interactive Elements - Hover effects, animations, and keyboard navigation support

Sections
1. Home- Introduction and quick links
2. About - Personal story, values, and languages
3. Skills - Technical skills with progress bars
4. Projects- Showcase of work including perfume business and IT projects
5. Education - Academic background and work experience
6. Contact - Contact information and message form
